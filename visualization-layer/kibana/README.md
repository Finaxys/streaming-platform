## Requirements

* Linux
* At least Java 7
* The command line wget

## How to install 

Download the Kibana 4.3.1 tar:
> ```ruby
> wget https://download.elastic.co/kibana/kibana/kibana-4.3.1-linux-x64.tar.gz
> ```

Extract it:
> ```ruby
> tar -xvf kibana-4.3.1-linux-x64.tar.gz
> ```

## How to run a Kibana

> ```ruby
> kibana-4.3.1-linux-x64/bin/kibana &
> ```
